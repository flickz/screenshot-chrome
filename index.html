Standardized Screenshot

A Very Opiniated One-click screen captures for Google Chrome

Install View on GitHub Download .zip Download .tar.gz

It will:

  • Resize the browser window to be 1280px wide.
  • Capture the current tab.
  • Add a OS X-ish title bar.
  • Add drop shadow to it.
  • Display the result in a new tab.
  • Resize the browser window to the original size.

Forked and heavily mutated from Full Page Screen Capture.

Install

You can get the extension on the Chrome Store.

Development

  1. Clone this repository.
  2. Open up Chrome and go to the chrome://extensions/ page (Window â†’ Extensions).
  3. Enable developer mode (if it's not already).
  4. Click on Load unpacked extension.
  5. Select the folder for this extension.

Keyboard Shortcut

Go to the bottom of the chrome://extensions/ page, click on Keyboard shortcuts and add the shortcut of your choice.
